Tentang DompetIn

Aplikasi web sederhana untuk membantu mahasiswa mengatur uang saku dan pengeluaran harian.

Kenapa Perlu Atur Budget?

Buat mahasiswa, uang saku atau gaji bulanan sering habis sebelum waktunya karena tidak tercatat dengan rapi. Dompetku membantu kamu mencatat setiap transaksi, menetapkan batas pengeluaran per kategori, dan melihat gambaran keuangan secara jelas.

  • Semua data tersimpan langsung di browser kamu (localStorage).
  • Tidak perlu koneksi internet atau server tambahan.
  • Cocok untuk latihan konsep HTML, CSS, dan JavaScript dasar.

Fitur Utama

  • Beranda — ringkasan saldo dan transaksi terbaru.
  • Transaksi — catat pemasukan/pengeluaran baru.
  • Anggaran — atur batas belanja per kategori.
  • Laporan — grafik dan rekap pengeluaran.
